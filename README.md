# CF-Photo-Service

## Quick Start

1. Install Dependencies:

```sh
$ npm install
```

2. Run worker locally:

```sh
$ npm run dev
# in browser, navigate to localhost:8787
```

---

## Deployment

```sh
$ npm run deploy
# in browser, navigate to https://cf-photo-service.<CF-ACCOUNT-NAME>.workers.dev/
```
